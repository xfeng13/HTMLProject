How to Upload/Download Files to/from your ASU www folder

 

You have two choices here:

 

Choice #1 (Using My ASU Service):

 

1. Logon to www.asu.edu, click on "My ASU", and use your ASURite ID and password to login

 

2. Once you logged in, from the left hand side, click on the brown box icon, then pick "My  Files" (see below)

3. You will need to re-enter your ASURite id and password to login and once logged in, you will see the following window on your screen

4. From the left hand side, under "Folders", double click on www folder to open it, files inside the www folder will be listed on the right hand side.

5. To upload a file inside the www folder, on top of the page, click on button "Browse" to select the file which you want to upload, then click on "OK" button to upload.

6. To download a file, click on the file directly, then pick "Save as" to save it on your local computer's hard drive.

 

Choice #2 (Using SSH):

 

1. If you use a home computer which does not contain a SSH software, logon to the following website        https://webapp3.asu.edu/swdownload/, Next click "Software Categories", from the list, pick "SSH", next  pick "SSH Windows" if you use a Windows system, follow the instruction to install it on your home computer. (Note: you need to remember where you install it, if you can't find it after installation, use search under Windows to search it)

 

2. Once you successfully install it on your computer, open it by picking "Secure File Transfer Client", the following windows show up, click on "Quick Connect"

 

 

3. Next type in your ASURite ID and password to login.

 

 

4. After you successfully login, you will see the following windows. The left panel show your files at C:, the right panel show your www folder. First you will need to click on www folder to open it, then you can upload/download a file by just dragging and dropping between any two folders. You also can delete a file from www folder (do some exercise on it).
