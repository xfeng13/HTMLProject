CSE180 FAQ on MyLab IT

Q1:  What is MyLab IT?

A: MyLab IT is an online interactive Microsoft Office 365 training program developed by Pearson - the textbook publisher, we will use the program to train on MS Excel and MS Word. Note: this interactive training program will simulate a MS Excel or Word running enviroment under either a Windows or a Mac PC, i.e. you need NOT to install MS Excel or Word on your PC to run it.

Q2:  Are there any system requirements to run MyLab IT in my home PC?

A:  Yes. Each client computer [workstation] running a MyLab IT must meet the minimum system requirements described at the following website:

https://support.pearson.com/getsupport/s/system-requirements

Q3:  Do I need to install any software to run MyLab IT?

A:  The answer is no. MyLab IT is light weighted and you need not to download any software! The training program will simulate a Microsoft Office running enviroment so that you can train on MS Office Word, Excel, etc on your own PCs without installing these softwares. But you do need to check the minimum system requirements and need to tune up your web browser.  

Q4:  Where can I find help if I have technical questions on running MyLab IT?

A:  Pearson provide 24/7 technical support for MyLab IT. Click on the following link to search MyITLab's FAQ or contact a Pearson's representative.

https://support.pearson.com/getsupport/s/learners

1. Contact Pearson's technical support via email, phone or chat.
2. Make sure you have an incident number.

Note: for questions posted on our course website's discussion board regarding MyLab IT, we will only be able to answer those which are directly related with training or course contents, sorry that we might not be able to help you to solve your PC's technical problem.

Q5:  I'm stuck on one of the training questions and can't go further, what shall I do?

You can click on a button called "Learning Aids" from the lower right corner of the training screen, then pick "Watch", it will bring up a demonstration video show exactly the steps on how to fulfill the task. You can also temperorily skip the question by pressing the next button, then later come back to solve the question.

Q6:  How many times can I take the the training for each chapter?

A:  You have unlimited times to do the trainings before the due date/time, so try your best to get 100% in each chapter. We will take your highest grade as your final grade for that chapter.

Q7:  How many questions in each chapter's training and how long does it take?

A:  Each MyLab trainig contains 10 ~ 15 questions. Depends on your skills, it may take 40 minutes to more than 2 hours to finish one. It took me about 50 minutes to finish Excel Chp.1 the first time, so we highly recommend that you reserve enough time for each chapter.  

Q8:  I didn't do well in one of the trainings and I'm worried about my score, what shall I do?

Each MyLab IT chapter is only worth 3% towards your final grade, so no need to worry too much if you didn't do well in one chapter, since you can take the 3 bonus quizzes to cover it up. On the other hand, try your best to get 100% in each chapter's training.

Q9:  Do I let those in charge know I have completed the homeworks in MyLab IT or will the program record and update my grade itself?

A:  For MyLab, you do not need to inform those in charge of your completed homeworks.  Just be sure to press the "Submit" button after the training, and a score will be automatically recorded on the MyLab IT gradebook (under tab "Result").

Q10:  Can I run MyLab IT on my Mac? (Notes to Mac users)

Yes, from above website you can see that MyLab IT does support Mac OS. We tested all traing questions by using a Mac in Coor 150, they all work! For few training questions (< 4), the demonstration video only shows the solution by using a PC's short cut keys, in such a case, you might need to use alternative methods, such as using the Ribbon to complete the task, this is because a Mac keyboard is different from a Windows keyboard.

To change your Mac keyboard to accept F1-F12 function keys like they do in Windows, click here for instructions.

Note: MyLab IT simulations are programmed to accept PC methods and keyboard shortcuts. When using the Mac solution, although using a Mac computer, the simulations will still operate as though on a PC due to how they are programmed.  Therefore, Mac-specific shortcuts, such as Control+Click to open the right-click menu, will not be recognized by the simulation. In such cases use alternative methods, such as those that use the Ribbon, to complete the activity.

Q12:  Can I install a Windows OS on my Mac and run MyLab IT from there (for Mac users)?

The answer is "Yes". Mac users, please be informed that you can use Boot Camp (a software already installed on a Mac) to run Windows on your Mac, i.e. install a dual-boot system. Be cautious that the installation procedure might be complicated for a beginner, if you plan to try it, click on the following website to find more information on it. 

https://support.apple.com/guide/mac-help/use-windows-on-your-mac-mh11850/mac
